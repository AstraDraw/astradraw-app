# Changelog

All notable changes to AstraDraw App (Frontend) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-25

### Added

**User Interface**

- Workspace dashboard with scene management
- Collection-based scene organization
- User profile with avatar upload
- Dark mode support throughout
- Quick search (Cmd+K) across workspaces
- Scene thumbnail previews

**Collaboration**

- Real-time collaboration with presence indicators
- Cursor tracking for collaborators
- Auto-collaboration for shared collection scenes
- End-to-end encryption for scene data

**Comments & Notifications**

- Threaded canvas comments anchored to elements
- @mentions in comments with autocomplete
- In-app notification bell with unread count
- Notifications page with timeline view

**Content Creation**

- Talktrack video recording with camera PIP
- Presentation mode using frames as slides
- Implicit laser pointer in presentation mode
- Custom pen presets (highlighter, fountain, marker)
- GIPHY integration for stickers and GIFs

**Authentication**

- OIDC/SSO login support
- Local authentication with email/password
- Session management with JWT cookies
